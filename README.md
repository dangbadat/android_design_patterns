# DesignPattern

####Java 23种设计模式

设计模式（Design pattern）是一套被反复使用、多数人知晓的、经过分类编目的、代码设计经验的总结。

设计模式分为三种类型，共23种：

 - **构建型模式**：单例模式、工厂模式、建造者模式、原型模式。
 - **结构型模式**：适配器模式、适配器模、桥接模式、装饰者模式、组合模式、外观模式、享元模式、代理模式。
 - **行为型模式**：模版方法模式、命令模式、迭代器模式、中介者模式、备忘录模式、解释器模式、状态模式、策略模式、职责链模式(责任链模式)、访问者模式。
